Detailed Description

The basic class which stores values of basic data structures like numbers,strings and boolean.

Some of the operations are similar for all datatypes, which have been inherired from PrimitiveDT. basic datatypes that appear as it is in an expression are stored as subclass of primitiveDT. eg. x=7+x; 7 is stored as primitive DT
